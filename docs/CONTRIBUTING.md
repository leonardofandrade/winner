# Contribuindo para o Winner

---

## 🧱 Regras Gerais
- Código em inglês
- Comentários em português
- Seguir arquitetura Winner

## 🧩 Telegram Bot
Contribuições para o bot devem seguir:

- Handlers finos
- Services com lógica
- Repositórios para acesso ao banco
- Runner simples
- Commands sem lógica
