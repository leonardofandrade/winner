# Contribuindo para o Winner

Obrigado por contribuir com o projeto Winner!  
Este documento explica como colaborar de forma consistente e organizada.

---

## 🧱 Regras Gerais

- Código sempre em **inglês**
- Comentários sempre em **português**
- Seguir arquitetura definida em `AGENT_GUIDE.md`
- Nunca colocar lógica em models, views ou commands
- Toda lógica deve estar em services
- Acesso ao banco sempre via repositories
